# WELCOME

## CLONE PROJECT

```
git clone git@github.com:sergioamr/img-api.git
```

## INSTALLATION

### Create a virtual environment to run this project

```
cd img-api

# Linux
sudo apt-get install python3-venv    # If needed
python3 -m venv .venv
source .venv/bin/activate

# macOS
python3 -m venv .venv
source .venv/bin/activate

# Windows
py -3 -m venv .venv
.venv\scripts\activate
```
