CURL

Example of uploading a picture using a command line script
```

#!/bin/bash test=`date "+PIC_%m%d%Y_%H%M%S.jpg"`
/system/sdcard/bin/getimage > $test

curl -X POST https://img-api.com/api/media/upload?key={{token}} -F "image=@./$test"

rm $test
```
